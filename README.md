# LiFT PV Visit Briefing Page

This folder is prepared as a simple GitHub Pages website.

## How to update

1. Edit `index.html` or replace the linked PDF files in this folder.
2. Preview `index.html` locally in a browser.
3. Run `.\publish.ps1 -Message "Update site"` from this folder.
4. GitHub Pages will publish the updated page automatically.

## Suggested GitHub Pages setting

Use GitHub Pages with:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

## Notes

- `index.html` is the live entry page.
- The original HTML file is kept as a source copy for reference.
- Please review confidential materials before making the repository public.
